Snow/Luxe: Custom Web Layout

Stisa, 16 December 2015

Hi everyone, this is just a quick post on a simple thing I found out today. When building for the web target with luxe, the default layout is just a black background with the game window in the middle. While this is okay while developing, I wanted to show a demo of the game on my website, so I wanted its layout to fit with the rest of the site. After a bit of digging (all right, a lot of digging, but only because I thought the layout was in flow, as it is the one responsible of building) I found the default layout inside snow lib folder, snow\flow\web\index.html, where snow is the path in which you installed snow.
